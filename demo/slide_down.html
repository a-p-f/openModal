Animated Modal

The only difference between this modal and our other alert modals is the addition of alertModal-slideDown.js and alertModal-slideDown.css.

Notice that if you , the window will not animate down again. alertModal-slideDown.js checks window.isFirstModalLoad(), and if false, moves the window into position without animation.

Notice that alertModal-slideDown.js patches window.closeModal(), so that it animates the alert off-screen before closing. This is our recommended approach to handling exit animations.

Lastly, note that the exit animation will only run if you explicitly call closeModal(). If you close the modal by clicking your browser's back button, the modal will be closed immediately.
